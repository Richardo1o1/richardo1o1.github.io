useScroll Memo:

  • 创建一个 ref,用于引用某个 DOM 元素(可能在JSX中通过 ref={ref} 绑定到一个元素上)。
  • 使用 useScroll 来监听这个被引用元素的滚动情况。
  • 设置滚动动画的触发范围:从元素开始进入视口底部(“0 1”)到元素完全离开视口底部 33% 的位置(“1.33 1”)。
  • 获取 scrollYProgress 值,这个值可能随着元素的滚动而变化,范围可能是 0 到 1。
